Instalação

3 passos, 3 minutos

Clone, instale e execute. Simples assim — sem configurações complexas ou dependências pesadas.

bash 
# ── 1. Clone o projeto ────────────────────────
git clone https://github.com/Thiago-code-lab/projeto-phoenix.git
cd projeto-phoenix

# ── 2. Instale as dependências ────────────────
pip install -r requirements.txt

# ── 3. Execute o app ──────────────────────────
streamlit run app.py

# ✅ Acesse http://localhost:8501
