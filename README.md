## 💡 O que é?

O Projeto Phoenix ajuda você a gerenciar todos os aspectos importantes da sua vida através de uma interface simples e intuitiva. Sem complicações, sem mensalidades, seus dados ficam com você.

### Por que usar?

✅ **Tudo em um lugar** — Pare de usar 10 apps diferentes  
✅ **Privado e seguro** — Seus dados ficam no seu computador  
✅ **Personalizável** — Adapte do seu jeito  
✅ **Gratuito** — 100% open source

## 🚀 Como usar

### Instalação rápida (3 passos)

```bash
# 1. Clone o projeto
git clone https://github.com/Thiago-code-lab/projeto-phoenix.git
cd projeto-phoenix

# 2. Instale as dependências
pip install -r requirements.txt

# 3. Execute o app
streamlit run app.py
```

Pronto! Acesse `http://localhost:8501` no seu navegador.
